### Запуск автотестов для разных языков интерфейса
#### Задание в рамках курса по автоматизации тестирования на Python + Selenium.

Необходимо реализовать решение, которое позволит запускать автотесты для разных языков пользователей, передавая нужный язык в командной строке.

##### Реализовано:

`conftest.py`

- обработчик, который считывает из командной строки параметр language
- логика запуска браузера с указанным языком пользователя
- time.sleep(30) для визуальной проверки работоспособности кода (например, для испанского языка текст на кнопке: "Añadir al carrito", для французского – "Ajouter au panier")

`test_items.py`

- тест, который проверяет, что страница товара на сайте содержит кнопку добавления в корзину (протестированы две кнопки: в карточке товара и под ней)
- тест запускается с параметром `language` командой:

```bash
  pytest --language=es test_items.py
```

##### Дополнительно

- гарантия работы кода для браузера Chrome

##### Структура 

- `README.md` – описание задания и проекта
- `conftest.py` – фикустуры pytest
- `test_items.py` – тест для проверки функциональности выбора языка
- `data.py` – тестовые данные
- `locators.py` – локаторы
